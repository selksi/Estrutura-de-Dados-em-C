# 📚 Estruturas de Dados 1 (EDA1)

Repositório dedicado ao armazenamento de exercícios, laboratórios e projetos desenvolvidos durante a disciplina de **Estruturas de Dados 1**, do curso de Engenharia de Software da **Universidade de Brasília (UnB)**.

O objetivo desta disciplina é compreender, implementar e analisar a eficiência de diferentes estruturas de dados e algoritmos de manipulação de memória utilizando a **linguagem C**.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **Linguagem:** C (Padrão C11/C99)
* **Compilador:** GCC
* **Ambiente de Desenvolvimento:** Linux Mint / VS Code
